HOW TO DEPLOY
=============

1. Go to cPanel > File Manager
2. Navigate to your staging site wp-content/
3. Upload each file to matching location (overwrite existing)
4. Visit any portal page once to trigger the DB migration
   (adds attachment_url column to page_comments table)

PAYMENT SETUP
=============

For the paid signup flow to work:
1. Go to WP Admin > Client Portal > Settings
2. Under "WHMCS Product -> Portal Plan Mapping"
3. Enter the WHMCS Product ID for each plan (briefstart, briefsite, etc.)
4. Make sure PayPal is configured as payment method in WHMCS

FEEDBACK IMAGE UPLOAD
=====================

In BriefBuilder feedback viewer:
- When dropping a pin, there's now an image attach button (camera icon)
- Click it to select an image (max 10MB)
- Image uploads immediately and shows a preview
- When you post the comment, the image is saved with it
- Images appear in the comments panel and can be clicked to view full size
